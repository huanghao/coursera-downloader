coursera-downloader
===================

Download videos and sub-scripts from coursera.org

$ coursera-dl.py eefun-002

$ coursera-dl.py --help

NOTES:
- It can only download those courses you involved in
- It uses chromedriver and your user/pass to simulate signin process
